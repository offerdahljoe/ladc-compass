# LADC Compass Product Vision

LADC Compass is not a generic information website. It is a clinical operating system, apprenticeship program, clinical mentor, documentation coach, and career companion for ADC-Ts, LADC interns, and LADCs.

## Product Rule

No random feature building. Every new feature must answer:

1. Where does this belong?
2. What problem does this solve?
3. What does it connect to?
4. How does it teach, organize, document, or grow the user?

## Five Pillars

- Learn: teach concepts.
- Think: teach clinical reasoning.
- Do: help perform the work.
- Document: help generate professional clinical language.
- Grow: help the user become a better counselor.

## Navigation

The primary products are:

- Today
- Client Journey
- Clinical Decision Navigator
- Group Studio
- Documentation Lab
- Medication Reference
- Calendar + Tasks
- Procentive Companion
- Reset Room
- Resource Library
- LADC Academy
- Licensure Journey
- Clinical Wisdom

## Golden Rule

Never build static information pages when a workspace would help more. Each page should help the user know what to do, why it matters, what to document, what to think about, what mistakes to avoid, and what comes next.

## Mantra

Simple on the surface. Powerful underneath.
