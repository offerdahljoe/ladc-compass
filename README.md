# LADC Compass

LADC Compass is a local learning portal for LADC internship organization. It is not a client management system and should not be used for real client identifying information.

## Open the Portal

The easiest version is the standalone file:

Double-click `LADCCompass.html`.

It opens directly in your browser and does not need a local server.

## Next.js Version

Double-click `Open-LADCCompass.cmd`.

Leave the command window open while you use the portal, then visit:

http://localhost:3000

If the page does not open automatically, copy that address into Chrome, Edge, or another browser.

## Cloud Access

The Next.js app is configured for cloud/static hosting. See `CLOUD-DEPLOYMENT.md`.

Cloud hosting lets you open the website from any computer. Supabase login and sync support has been added for saved entries once you add your Supabase keys.
